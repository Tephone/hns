**タイトル**  
HNS（horse networking service）

**コンセプト**  
ライダーによるライダーのためのソーシャルネットワークサービスアプリ

**バージョン**  
ruby2.6.5  rails 5.2.4

**機能一覧**
* ログイン機能
* ユーザー登録機能
* 文章投稿機能
* 文章編集機能
* 文章削除機能
* 画像投稿機能
* 画像削除機能
* 画像一覧表示機能
* 投稿お気に入り機能
* 投稿コメント機能
* 運動日誌機能
* ゲストログイン機能
